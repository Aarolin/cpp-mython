## Проект Интерпретатор Mython ##

### Описание проекта ###
Данный проект посвящен отработке навыков полученных в ходе обучения на курсе Практикума "Разработчик C++".
Mython - упрощенное подможество языка программирования Python. Ниже приведено описание синтаксиса Mython.

### Описание синтаксиса: ###
### Примитивные типы данных в Mython: ###
Mython поддерживает следующие типы данных:
Числа - используются только целые числа. С ними можно выполнять арифметические операции: сложение, вычитание, умножение, деление. 
Строки - последовательность символов ограниченная одинарными или двойными кавычками.
Логические константы - значения True и False.
None - специальное значение, аналог nullptr в C++.

### Идентификаторы: ###
Используются для имен переменных и классов. Начинаются со строчной или заглавной латинской буквы либо со знака нижнее подчеркивание,
после чего может идти произвольная последовательность букв, цифр и нижних подчеркиваний.

### Создание классов: ###
Mython можно создавать свои типы данных. Класс имеет собственные поля и методы. Объявление класса начинается с ключевого слова
"class", за которым следует идентификатор имени, двоеточие, объявление методов. Поля не объявляются заранее, а добавляются во время первой
инициализации. Специальный метод __init__ играет роль конструктора класса (является необязательным).

### Наследование: ###
В Mython у класса может быть один родительский класс. Указывается в скобках после имени класса.

### Типизация: ###
Mython - язык с динамической типизацией, это значит, что тип переменной определяется во время исполнения программы и может меняться в ходе её работы.
Благодаря этому указывать тип данных при первом использовании переменной не требуется.

### Операции: ###
В Mython определены:
Арифметические операции для целых чисел, деление выполняется нацело. Деление на ноль вызывает ошибку времени выполнения.
Операция конкатенации строк, например: s = 'hello, ' + 'world'.
Операции сравнения строк и целых чисел ==, !=, <=, >=, <, >; сравнение строк выполняется лексикографически.
Логические операции and, or, not.
Унарный минус.

### Приоритет операций (в порядке убывания приоритета): ###
Унарный минус.
Умножение и деление.
Сложение и вычитание.
Операции сравнения.
Логические операции.
Порядок вычисления выражений может быть изменён скобками

### Комментарии: ###
Mython поддерживает однострочные комментарии, начинающиеся с символа #.

### Функция str: ###
Преобразует переданный ей аргумент в строку. Если аргумент является объектом класса, то вызывает у него специальный метод __str__.
Если такой метод у класса отсутсвует, то возвращает представление адреса объекта в памяти.

### Команда print: ###
Принимает набор аргументов, разделенных запятой, печатает их в стандартный вывод и дополнительно выводит перевод строки.

### Циклы: ###
В Mython отсутсвуют циклы, но в качестве замены доступна рекурсия у методов.

### Сборка CMake-ом: ### 
> 0. Создайте папку для сборки программы .
> 1. Перейдите в папку сборки с помощью консоли и введите: `cmake <путь к CMakeLists.txt>`
> 2. Введите команду `cmake --build .`
> 3. После сборки появится исполняемый файл mython.exe, запустив который можно начать писать код на языке Mython

### Системные требования: ###
C++ 17  
GCC (MinGW-w64) 12.2.0
